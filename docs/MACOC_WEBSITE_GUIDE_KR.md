# MACOC 웹사이트 관리 가이드

**Musical Arts Competition of Orange County**
**웹사이트:** https://www.musicalartsoc.org

---

## 이 문서는 무엇인가요?

MACOC 웹사이트가 새롭게 바뀌었습니다.
기존에 사용하던 **Wix + Jotform** 대신, 더 저렴하고 관리하기 쉬운 방식으로 전환했습니다.

이 문서는 세 가지를 설명합니다:

1. **웹사이트 내용(Contents)** 을 어떻게 바꾸는지
2. **등록 양식(Registration Form)** 이 어떻게 작동하고, 등록 현황을 어떻게 보는지
3. **결제(Billing)** 가 어떻게 처리되는지

---

## 1. 웹사이트 내용 (Contents) 관리

### 무엇을 바꿀 수 있나요?

| 항목 | 예시 |
|------|------|
| 대회 날짜 | "Saturday, May 16, 2026" |
| 대회 장소 | "California State University, Long Beach" |
| 등록 기간 | 시작일, 마감일, 늦은 등록 마감일 |
| 수상자 콘서트 | 날짜, 장소, 시간 |
| 참가비 | Membership $40, Entry $50, Late Fee $70 |
| 연락처 | 이메일, Division별 담당자 |
| 역대 수상자 | 연도별 수상자 목록 |
| 교사 명부 | 악기별 교사 목록 |

### 어떻게 바꾸나요?

**방법: Google Sheets에서 수정**

Google Sheets(구글 스프레드시트)에 모든 내용이 정리되어 있습니다.
엑셀처럼 생긴 온라인 표에서 내용을 수정하면, 웹사이트에 자동으로 반영됩니다.

```
수정 순서:

1. Google Sheets를 연다
2. 바꾸고 싶은 칸을 찾는다
3. 내용을 수정한다
4. 저장은 자동으로 된다
5. 웹사이트가 자동으로 업데이트된다
```

**예시: 대회 날짜를 바꾸고 싶을 때**

| Key | Value |
|-----|-------|
| competition.date | Saturday, May 16, 2026 |

위 표에서 "Saturday, May 16, 2026" 부분만 새 날짜로 바꾸면 됩니다.

**예시: 올해 수상자를 추가하고 싶을 때**

Winners 탭에서 새 행을 추가하고, 연도/부문/등수/이름을 입력하면 됩니다.

> **핵심:** Wix처럼 복잡한 편집기를 배울 필요가 없습니다.
> 구글 스프레드시트(엑셀과 비슷)만 수정하면 됩니다.

## 2. 등록 양식 (Registration Form) 관리

### 등록 과정은 어떻게 되나요?

학생이 대회에 등록하는 과정은 3단계입니다:

```
Step 1: 학생 & 교사 정보 입력
   - 교사: 이름, 이메일, 전화번호
   - 학생: 이름, 이메일, 생년월일, 나이, 주소
   - 나이 증명 서류 업로드 (출생증명서 또는 여권)

Step 2: 연주곡 정보 입력
   - 부문 선택 (Piano 등)
   - Section 선택 (나이별)
   - 연주곡 2개: 곡명, 작곡가, 시대

Step 3: 확인 & 제출
   - 입력한 내용 확인
   - 제출 버튼 클릭
   - 결제 페이지로 이동
```

### 등록 현황은 어디서 보나요?

**Google Sheets에서 바로 볼 수 있습니다.**

누군가 등록을 하면, Google Sheets에 자동으로 한 줄이 추가됩니다.

| 시간 | 결제상태 | 교사 이름 | 학생 이름 | 부문 | Section | 연주곡 |
|------|---------|----------|----------|------|---------|--------|
| 2026-03-20 | Pending | Kim, Jane | Park, David | Piano | Section III | Sonata... |
| 2026-03-21 | Pending | Lee, Sarah | Choi, Emily | Piano | Section V | Prelude... |

이 표에서 한눈에 확인할 수 있는 것들:
- **누가 등록했는지** (학생 이름, 교사 이름)
- **언제 등록했는지** (날짜와 시간)
- **돈을 냈는지** (결제 상태: Pending = 아직, Paid = 완료)
- **어떤 부문인지** (Piano, Strings 등)
- **어떤 곡을 연주하는지**
- **나이 증명 서류** (링크 클릭하면 바로 볼 수 있음)

> **핵심:** Jotform에 로그인해서 찾아볼 필요가 없습니다.
> 평소 쓰는 Google Sheets에서 모든 등록 정보를 바로 볼 수 있습니다.

### 나이 증명 서류는 어디에 저장되나요?

학생이 업로드한 출생증명서나 여권 사진은 **Cloudinary**라는 안전한 파일 저장소에 보관됩니다.
Google Sheets에 있는 링크를 클릭하면 바로 확인할 수 있습니다.

## 3. 결제 (Billing) 관리

### 결제 과정은 어떻게 되나요?

```
1. 학생이 등록 양식을 작성하고 제출한다
2. "결제하기(Complete Payment)" 버튼이 나타난다
3. 버튼을 클릭하면 Stripe 결제 페이지로 이동한다
4. 학생(또는 학부모)이 카드 정보를 입력하고 결제한다
5. 결제가 완료되면 Google Sheets에 기록된다
```

### Stripe란 무엇인가요?

Stripe는 전 세계에서 가장 많이 쓰이는 온라인 결제 서비스입니다.
Amazon, Google 같은 대기업도 사용합니다. 안전하고 믿을 수 있습니다.

### 결제 현황은 어디서 보나요?

두 곳에서 확인할 수 있습니다:

1. **Google Sheets** - 등록 목록에서 "결제 상태" 열 확인
2. **Stripe Dashboard** (stripe.com) - 결제 금액, 날짜, 상세 내역 확인

※ Stripe 카드 결제 수수료(2.9% + $0.30)는 기존 Wix와 동일하게 거래 시에만 발생합니다.

## 관리가 얼마나 더 쉬워지나요?

### 기존 방식의 불편함

| 문제 | 상세 |
|------|------|
| Wix 편집기 사용 | 로그인 후 복잡한 편집기에서 내용 수정 |
| Jotform 별도 관리 | 등록 양식은 Jotform에서 따로 관리 |
| 데이터 분산 | 웹사이트는 Wix, 양식은 Jotform, 결제는 또 따로 |
| 등록 현황 확인 어려움 | Jotform에 로그인해서 제출 내역 하나씩 확인 |

### 새로운 방식의 편리함

| 장점 | 상세 |
|------|------|
| Google Sheets 하나로 관리 | 내용 수정, 등록 현황, 결제 상태 모두 한 곳에서 확인 |
| 엑셀처럼 쉬운 수정 | 칸을 클릭하고 내용만 바꾸면 끝 |
| 실시간 확인 | 누가 등록했는지 바로 보임 |
| 여러 사람이 동시에 보기 가능 | Google Sheets 공유로 여러 담당자가 함께 확인 |
| 데이터 다운로드 쉬움 | Google Sheets에서 엑셀 파일로 바로 다운로드 |

### 한눈에 비교

```
              기존 (Wix + Jotform)          새 시스템
───────────────────────────────────────────────────────
비용          $128/월 ($1,536/년)           $0/월 (~$12/년)
내용 수정     Wix 편집기 (복잡)             Google Sheets (쉬움)
등록 관리     Jotform 로그인 필요           Google Sheets에서 바로 확인
결제 관리     별도 확인 필요                Stripe + Google Sheets
데이터 위치   Wix + Jotform (분산)          Google Sheets (한 곳)
공유          각각 로그인 필요              Google Sheets 링크 공유
```

## 자주 묻는 질문

**Q: Google Sheets를 잘 모르는데 괜찮을까요?**
A: 엑셀을 사용해 보셨다면 거의 같습니다. 칸을 클릭하고 내용을 입력하면 됩니다.

**Q: 웹사이트가 갑자기 안 되면 어떡하나요?**
A: GitHub Pages는 매우 안정적인 서비스입니다. 문제가 생기면 개발 담당자에게 연락하세요.

**Q: 등록 양식을 바꾸고 싶으면요?**
A: 양식의 항목(질문)을 바꾸려면 개발 담당자에게 요청해 주세요. 간단한 수정은 빠르게 가능합니다.

**Q: 결제된 돈은 어디로 들어오나요?**
A: Stripe 계정에 연결된 은행 계좌로 자동 입금됩니다. Stripe Dashboard에서 확인할 수 있습니다.

**Q: 여러 부문(Strings, Voice 등)도 등록할 수 있나요?**
A: 현재는 Piano 부문만 가능하며, 나머지 부문은 준비 중입니다.
